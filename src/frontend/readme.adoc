== SkillMatrix Frontend

Install all required packages:
....
$ cd src/frontend/ 
$ npm ci
....
To run all tests:
....
$ npm test
....
To create a build instead, run:
....
$ npm run build
....
The _package-lock.json_ contains ESLint. To run ESLint on all files in `src/frontend/src`:
....
./node_modules/.bin/eslint src/ --ext .js
....
